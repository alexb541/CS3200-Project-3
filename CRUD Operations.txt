Team Leaderboard:


Create:

# adds teams with a starting score of 0
ZADD leaderboard:event:2025gpr 0 2228 0 254 0 1678 0 971


Read:

ZREVRANGE leaderboard:event:2025gpr 0 9 WITHSCORES     # top 10 teams
ZREVRANGE leaderboard:event:2025gpr 0 -1 WITHSCORES    # full ranking
ZSCORE    leaderboard:event:202gpr 2228                # one team's score
ZREVRANK  leaderboard:event:2025gpr 2228                # one team's rank (0-indexed)
ZCARD     leaderboard:event:2025gpr                     # number of teams tracked


Update:

# Incremental: a new scout report adds 45 points to team 2228's total
ZINCRBY leaderboard:event:2026casf 45 2228

# Absolute: recomputed average from MongoDB
ZADD leaderboard:event:2026casf 87.3 2228


Delete:

ZREM leaderboard:event:2026casf 2228     # remove one team (e.g., team withdrew)
DEL  leaderboard:event:2026casf          # wipe the entire leaderboard for that event
